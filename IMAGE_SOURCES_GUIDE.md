## ✅ 下载步骤

1. **点击上方链接**，进入 Unsplash/Pexels 搜索页面
2. **选择高质量图片：**
   - 分辨率高于目标尺寸（至少 +20%）
   - 专业摄影水准
   - 符合品牌调性（明亮、专业）
3. **下载原图：**
   - Unsplash：点击 "Download free" → 选 "Original" 或高分辨率
   - Pexels：点击 "Free Download" → 选最大尺寸
4. **检查许可证：**
   - Unsplash License: 免费商用，无需署名（可署名但不是必须）
   - Pexels License: 免费商用，无需署名
   - **确认无误即可使用**

## 🚀 快速下载（推荐方式）

我们准备了自动化脚本，一键下载预选的图片。

### 方式一：使用下载脚本（最快）

```bash
cd finance-site
node scripts/download-photos.js
```

脚本会自动：
1. 从 `photos/manifest.json` 读取 10 张推荐图片的 Unsplash 链接
2. 下载到 `photos/` 对应子目录
3. 自动创建目录结构

**如果网络不稳定或某些图片下载失败：**
- 手动打开 `photos/manifest.json` 中的 URL，直接在浏览器中右键下载
- 或替换为其他 Unsplash/Pexels 图片链接（格式需匹配）

### 方式二：手动下载（自由选择）

按照上方搜索链接，在 Unsplash/Pexels 中找到满意图片后：

1. 点击 "Download free"
2. 选择 "Original" 或最高分辨率
3. 保存到对应目录：
   - 团队照片 → `photos/team/`
   - 办公室 → `photos/office/`
   - 计算器 → `photos/general/`
   - 产品图 → `photos/products/`

## 🔧 后期处理（可选）

### 1. 裁剪到精确尺寸
使用 [Squoosh.app](https://squoosh.app) 在线工具：
- 上传图片
- 选择 "Resize" 操作
- 输入目标尺寸（如 1200x500）
- 调整裁剪位置
- 导出为 JPG，质量 80-85%

### 2. 品牌色调调整（可选）
如果需要统一色调：
- 使用 Photoshop/GIMP
- 添加半透明金色渐变叠加（#d4a855，透明度 10-15%）
- 或使用 Canva 模板快速处理

### 3. 压缩优化
Squoosh 已自动压缩，也可用：
- [ImageOptim](https://imageoptim.com) (Mac)
- [TinyPNG](https://tinypng.com) (在线)

目标文件大小：100-200KB

## 📂 文件放置

下载的图片应先放在 `photos/` 目录。完成后，运行替换脚本复制到网站：

```bash
cd finance-site
node scripts/replace-photos.js
```

该脚本会从 `photos/` 复制到 `public/images/` 对应位置。

或者手动复制：

```bash
# 团队照片
cp photos/team/*.jpg public/images/team/
# 办公室照片
cp photos/office/office-main.jpg public/images/office/
# 计算器图片
cp photos/general/mortgage-dashboard.jpg public/images/
# 产品图片
cp photos/products/*.jpg public/images/products/
```

## 📝 注意事项

✅ **版权安全：** 仅使用 Unsplash/Pexels/Pixabay 等明确可商用的图库
✅ **命名规范：** 使用小写 + 连字符，与现有文件名一致
✅ **尺寸标准：** 至少达到要求尺寸，避免模糊
✅ **风格统一：** 优选现代、专业、明亮色调的图片
✅ **墨尔本元素：** 寻找包含澳洲/墨尔本特色的图片更佳

❌ **避免：**
- 带水印的图片
- 需要署名的图片（虽然允许，但影响美观）
- 低分辨率图片（<1000px 宽度）
- 风格过于夸张或随意的图片

## 📋 自定义图片链接（高级用法）

如果想使用自己的图片选择，编辑 `photos/manifest.json`：

```json
{
  "team": [
    { "filename": "steven-mo.jpg", "url": "https://unsplash.com/photos/你的图片ID" }
    // ...
  ]
}
```

支持来源：
- Unsplash: `https://unsplash.com/photos/{id}`
- Pexels: `https://www.pexels.com/photo/{id}/...`

修改后重新运行 `node scripts/download-photos.js` 下载。

## 🆘 遇到问题？

| 问题 | 解决方案 |
|------|----------|
| 下载失败 | 检查网络，手动打开 URL 下载，或替换 manifest 链接 |

| 页脚仍为白色 | 确保已将 `bg-primary-950` 改为 `bg-primary-900` |
| 部署后旧图缓存 | Vercel -> Deployments -> 选最新 -> ... -> Purge Cache |
